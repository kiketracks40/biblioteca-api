# Database Setup

## Restaura la Base de datos
1. Abre SQL managment Studio
2. Click derecho en Base de datos > Restaura Base de datos
3. Selecciona la fuente: Dispositivo > Browse > [select .bak file]
4. Clicl Ok para restaurar

## Or Inicia Scripts
1. Abre SQL Server Management Studio
2. Abre el Archivo Script (biblioteca_script.sql)
3. Ejecuta script to para crear la base de datos y cargar datos.
